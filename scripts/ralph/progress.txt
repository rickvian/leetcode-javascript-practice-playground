## Codebase Patterns
- Jest configuration: Use default jest config in package.json scripts
- LeetCode file naming: [number]-[problem-name].js (e.g., 0001-two-sum.js)
- Function exports: Use module.exports for single functions or { func, Class } for multiple exports
- LinkedList operations: Create helper ListNode class for linked list problems
